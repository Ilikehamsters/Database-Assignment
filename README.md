Tutorial on how to run:
1. Unzip the zip file into the "MAMP\htdocs" folder.
2. Unzip the "COMP1044_SRC" folder if neccessary.
3. Go to PHPMyAdmin and create a "utf8_general_ci" database collation named "comp1044_database".
4. Import the "COMP1044_database.sql" into the database.
5. Run the webpage by going to "http://localhost/COMP1044_CW_G16/COMP1044_SRC/Login_Page.php". The actual path may difer depending on the exact location of the (Login_Page.php).

Additional Notes:
1. The default password (which leads to the password changing page) is "defaultPWD".
2. The passwords of all the accounts can be viewed in the "user_login" table of the database, if you have troubles trying to log in to the sample data.
